CSS字体样式属性

Font-size

用于设置字号

Font-family

用于设置字体，常见的有宋体，微软雅黑…

字体字号使用技巧

1.网页中普遍使用14px

2.尽量使用偶数的字号，ie6等老式浏览器用奇数有bug

3.多个字体都要用英文都好隔开

4.中文字体加引号，英文一般不用

5.如果英文字体有空格,$,#等符号要加引号或单引号.

6.尽量使用系统默认字体

Css Unicode 字体

兼容性更好

尽量写Unicode字体：宋体和微软雅黑

Css注释

Font-weight 字体加粗

Font-style

normal:指定字体样式为正常字体

italic:指定文本样式为斜体

Font综合设置字体样式

Chrome开发者工具(F12)

1.ctrl+滚轮可以放大缩小

2.左边是html，右边是css

3.右边css样式可以改变数值和颜色查看改变后的效果，光标在字号上,（按键盘上的上下键可以调大小）

选择器

标签选择器(元素选择器)

用html标签名作为选择器，按标签名进行分类，为网页中某一类标签指定统一的css样式

最大的优点能为网页中同类标签统一样式，同时也i是其缺点，不能设计差异性样式

类选择器

类选择器用英文点好(.)来进行标识,后面紧跟类名

优点在于可以为元素对象定义单独或相同的样式

Tips：

  • 长名词或词组可以使用中横线为选择器命名
  • 不建议使用_为选择器命名(存在兼容性问题)
  • 不要用纯数字和中文命名

多类名选择器

1.样式显示效果跟HTML元素中的类先后顺序没有关系，受css样式书写上下顺序有关。

2.各个类名之间用空格隔开。

ID选择器

id选择器和类选择器的区别

类选择器是可以多次使用的，id选择器不可重复使用。

通配符选择器

*指所有的标签，很少使用

伪类选择器

链接伪类选择器

为了向某些选择器添加特殊效果。

  • :link 未访问的链接

  • :visited 已访问过的链接

  • :hover 鼠标移到链接上

  • :active 选定的链接，即点击但不松开鼠标，顺序不可以乱调 lvha

结构伪类选择器

  • :first-child 选取属于父元素的首个子元素的指定选择器

  • :last-child 选取属于父元素的首个子元素的指定选择器

  • :nth-child(n) 匹配属于父元素的N个子元素，不论元素类型，n=even 指选偶数 n=odd指选奇数 n=n指从开始选 。n还可以用公式表示（如2n 选偶数）都是从第一个数开始数。

  • :nth-last-child(n) 选择器匹配属于其元素的第N个子元素的每个元素，不论元素类型，从最后一个元素开始计数。 n可以是数字，关键词，公式。

目标伪类选择器

CSS外观属性

color：文本颜色

1.预定义颜色：red，green

2.十六进制，如#FF0000(#F00),#FF6600(#F60),#29D794 (常用)

3.RGB代码，如红色:rgb(255,0,0)或rgb(100%,0%,0%)

使用rgb代码颜色百分比值，取0不可省略%。

line-height：行间距

行与行之间的距离，单位常用像素px，相对值em,百分比%。

一般行间距比字号大7,8像素即可。

text-align：水平对齐方式

left：左对齐(default) right：右对齐 center：居中对齐

text-indent ：首行缩进

单位em ：1em只一个字的宽度，如果是汉字，1em就是1个汉字的宽度

letter-spacing：字间距

word-spacing：单词间距（对中文无效）

颜色半透明（CSS3）

文字阴影

引入CSS样式表（书写位置）

行内式（内嵌式）

通过标签的style属性来设置元素的样式

内部样式表

内嵌式是将CSS代码集中写在html文档的head头部标签里，并用style标签定义

外部样式表（外链式）

指将所有的样式放在一个或多个一.css为扩展名的外部样式表文件中

标签显示模式

块级元素

每个元素通常都会独自占据一整行或多整行，可以对其设置宽度，高度，对齐等属性

特点：

1.总是从新行开始

2.宽度，行高，边距以及内边距都可以控制

3.宽度默认是容器的100%

4.可以容纳内敛元素和其他块元素。

行内元素

行内元素（内联元素）不占单独的区域，仅仅靠自身的字体大小和图像尺寸来支撑结构，一般不可以设置宽度，高度，对齐等属性，常用于控制页面文本的样式

特点：

1.和相邻的行内元素在一行上显示

2.宽，高是无效的 但水平方向上的padding值和margin可以设置，垂直方向无效

3.默认宽度是其内容宽度

4.行内元素只能容纳文本或其他行内元素（a特殊）

注意：

1.只有文字才能组成段落 因此p里面不能放块级元素，同理还有h1~h6,dt,都是文字块级标签，里面不可以放块级元素

2.链接里不可以放链接。

行内块元素（inline-block）

在行内元素中有几个特殊的标签，可以对其设置宽度和对齐属性

特点：

1.和相邻行内元素（行内块）在一行上，但是之间会有空白间隙

2.某人宽度就是本身内容的宽度

3.宽度，行高，边距以及内边距都可以控制

标签模式转换 display

复合选择器

交集选择器

交集选择器有两个选择器组成，用.连接，第一个是标签选择器，第二个是class选择器（或id选择器—不推荐），两个选择器之间不能有空格。 如p.special

Tips：

交集选择器是并且的意思

用的相对较少

并集选择器

并集选择器（css选择器分组）是各个选择器通过逗号连接而成的，任何形式的选择器（包括标签选择器，class选择器，id选择器等），都可以作为并集选择器的一部分。

Tips：

并集选择器的意思是：只要逗号隔开的所有选择器都会执行后面的样式

后代选择器

后代选择器又称包含选择器，用来选择元素或元素组的后代，其写法就是把外层标签写在前边，内层标签写在后边，中间用空格分隔。当标签发生嵌套时，内层标签成为外层标签的后代。

子元素选择器
