CSS字体样式属性

Font-size

用于设置字号

Font-family

用于设置字体，常见的有宋体，微软雅黑…

字体字号使用技巧

1.网页中普遍使用14px

2.尽量使用偶数的字号，ie6等老式浏览器用奇数有bug

3.多个字体都要用英文都好隔开

4.中文字体加引号，英文一般不用

5.如果英文字体有空格,$,#等符号要加引号或单引号.

6.尽量使用系统默认字体

Css Unicode 字体

兼容性更好

尽量写Unicode字体：宋体和微软雅黑

Css注释

Font-weight 字体加粗

Font-style

normal:指定字体样式为正常字体

italic:指定文本样式为斜体

Font综合设置字体样式

Chrome开发者工具(F12)

1.ctrl+滚轮可以放大缩小

2.左边是html，右边是css

3.右边css样式可以改变数值和颜色查看改变后的效果，光标在字号上,（按键盘上的上下键可以调大小）

选择器

标签选择器(元素选择器)

用html标签名作为选择器，按标签名进行分类，为网页中某一类标签指定统一的css样式

最大的优点能为网页中同类标签统一样式，同时也i是其缺点，不能设计差异性样式

类选择器

类选择器用英文点好(.)来进行标识,后面紧跟类名

优点在于可以为元素对象定义单独或相同的样式

Tips：

  • 长名词或词组可以使用中横线为选择器命名
  • 不建议使用_为选择器命名(存在兼容性问题)
  • 不要用纯数字和中文命名
