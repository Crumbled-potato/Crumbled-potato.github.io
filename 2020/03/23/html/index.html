标签属性

1.标签可以有多个属性，必须写在开始标签中，位于标签后面

2.属性之间不分先后顺序

3.任何标签都有默认值，不写则属性为默认值

Name 属性

如果要正确地被提交，每个输入字段必须设置一个 name 属性。

表格属性

属性名 含义 常用属性值
border 设置表格的边框（默认为0） 像素值
cellspacing 设置单元格与单元格边框之间的空白间距 像素值（默认为2）
cellpadding 设置单元格内容与单元格边框之间的空白间距 像素值（默认为1）
width 设置表格的宽度 像素值
height 设置表格的高度 像素值
align 设置表格在网页内中的水平对齐方式 left center right

表单

表单域 —存放表单的区域

<form > 用来定义表单

Action 属性

action 属性定义在提交表单时执行的动作。

向服务器提交表单的通常做法是使用提交按钮。

通常，表单会被提交到 web 服务器上的网页。

Method 属性

method 属性规定在提交表单时所用的 HTTP 方法（GET 或 POST）：

name属性

当有多个表单时可以通过name来进行区分

<input> 单标签

1.可通过type属性来变换

2.value为自定义默认值

3.maxlength(正整数)属性

4.html5新增属性：

属性 用法 含义
placeholder <input type=”text” placeholder=”请输入用户名”> 占位符
autofocus <input type=”text” autofocus=”” > 自动获得焦点
multiple <input type=”file” multiple=””> 多文件上传
autocomplete <form action=””><br/> <input type=”text” autocomplete=”” name=”username”> <input type=”submit” value=”submit”> </form> 自动记录完成 （要有提交按钮 要有name值 off 关闭 _on_打开）
required <input type=”text” required=””> 不能为空
